ampersand

Assignments made easy.

Ampersand takes the work out of assigning work. Just create an assignment, fill in a name and description, and you're done. It's that simple. And Google Drive integration makes the process of submitting work foolproof and seamless.

Create an assignment!

What is it?

Ampersand allows teachers to assign work to students. Each student chooses a file from Google Drive to share as their response, and then they're able to see all the work submitted by others. All you need is a Google account, and it's completely free for basic use — forever.

Can I use it?

Yes! It was created for AFE in Santa Cruz, California, but anyone can use it for any (legal) purpose. If you encounter any issues, or just want to talk, don't hesitate to contact the author!

How?

Click here to create a new assignment. Fill in a name and description (HTML allowed), and then give the link to anyone you want responses from. Your assignment will never expire.
